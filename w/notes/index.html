2023.02.20

windows contextual search

The only feature I really want to be added to Windows is some sort of contextual search. I wish I could just type in "emails about [topic]" or "show me pictures from last Christmas", and have Windows show me files based on the context of my search. I know that this feature has been implemented in many other apps (E.G.: Google Photos), so I'm surprised that it hasn't been implemented already. It doesn't even require internet access to do this, so there shouldn't be any privacy concerns.

An old version of the Microsoft Photos app on Windows 10 had a feature where it would identify elements of your photos and tag them, so you could search for things like "Christmas" or "pets". Unfortunately they removed that feature in the Windows 11 version, which really seems like a step back to me. It processed your photos locally and there was no internet connection necessary for that feature to work, which is a novelty these days. I don't know why they removed it.

Anyway, just some of my thoughts on the topic.
